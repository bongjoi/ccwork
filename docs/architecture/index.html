  deactivate Api

  deactivate Ctx
  Editor->>Editor: onDone() · setSaving(false)
  deactivate Editor

4. 상태 전이

App.tsx의 selectedNoteId(nullable id) × isCreating(boolean) 조합이 화면 모드를 결정한다. 

stateDiagram-v2
  [*] --> Idle
  Idle: Idle (빈 화면)
  Editing: Editing (기존 노트 편집)
  Creating: Creating (새 노트 작성)

  Idle --> Editing: handleSelectNote(id)
  Idle --> Creating: handleNewNote()
  Editing --> Editing: handleSelectNote(otherId)
  Editing --> Creating: handleNewNote()
  Creating --> Editing: handleSelectNote(id)
  Editing --> Editing: handleDone() (선택 유지)
  Creating --> Idle: handleDone()

  note right of Idle
    selectedNoteId = null
    isCreating = false
  end note
  note right of Editing
    selectedNoteId = string
    isCreating = false
  end note
  note right of Creating
    selectedNoteId = null
    isCreating = true
  end note
